# Mon-Portfolio-Cybersecurite
J'ai développé un portefeuille de projets de cybersécurité rigoureux sur des clients fictifs couvrant NIST, les audits, Linux, SQL, les actifs, les menaces, les vulnérabilités, la détection, la réponse aux incidents, l'escalade, Wireshark, tcpdump, IDS (Suricata), SIEM (Splunk, Chronicle) et l'automatisation Python.
