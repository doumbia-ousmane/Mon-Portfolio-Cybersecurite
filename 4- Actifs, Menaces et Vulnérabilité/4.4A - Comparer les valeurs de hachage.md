# Comparer les valeurs de hachage

## Description
En tant qu'analyste de sécurité, le hachage est l'une des mesures de sécurité que nous pouvons mettre en œuvre. Il génère un code indéchiffrable. Son fonctionnement repose sur l'identification unique du contenu d'un fichier, identifié par un identifiant unique (valeur de hachage ou condensé). Par exemple, un programme malveillant peut imiter un programme original. Si une seule ligne de code diffère, la valeur de hachage produite est différente. Les équipes de sécurité peuvent alors identifier le programme malveillant et prendre des mesures pour atténuer les risques.
